# Catalist — Telegram-бот

Бот для приёма заявок на дотацию ветеринарной помощи уличным кошкам.

## Требования

- Python 3.10+
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))

## Установка

```bash
cd catalist-bot
pip install -r requirements.txt
```

## Настройка

Скопируйте `.env.example` и заполните:

```bash
cp .env.example .env
```

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от BotFather |
| `ADMIN_CHAT_IDS` | ID администраторов через запятую (узнать свой ID: [@userinfobot](https://t.me/userinfobot)) |

## Запуск

```bash
export BOT_TOKEN="your-token"
export ADMIN_CHAT_IDS="123456789"
python bot.py
```

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветствие |
| `/apply` | Подать заявку на дотацию |
| `/help` | Информация о программе |
| `/cancel` | Отменить текущую заявку |

## Процесс заявки

1. Пользователь вводит `/apply`
2. Бот последовательно запрашивает: фото, процедуру, стоимость, ФИО, телефон, почту, комментарий
3. Пользователь подтверждает данные
4. Заявка сохраняется в SQLite и отправляется администраторам
5. Администратор одобряет (с указанием суммы) или отклоняет
6. Пользователь получает уведомление о решении
7. При одобрении — дополнительное сообщение с предложением поддержать проект
